Welcome to Warehouse Project!

The Warehouse application creates a data model for a simple Warehouse management system. A Warehouse is located at a single address and can hold many stock items at its location. A single stock item can identify itself with only one type of product however a Warehouse can have multiple stock items of a certain product. Then each product type can be marked by multiple tags.

We have implemented this simple Warehouse data model in a Play application and more about the data model can be seen by the entity relationships below.
